FM Analytics is a webapp that allows players of the Football Manager video game series to store their squad data as they journey through their save, tracking and analysing players' performances over seasons.

Using the app

  • To use the app, it is important that the squad view is properly formatted for processing. Subscribe to the associated squad view on Steam.
  • At the end of every season (or after the last match of the season - around May/June in England), navigate to the squad view and load the FM Analytics Squad View. It is recommended that you set a yearly reminder (by creating a note with a recurring reminder) to make sure you don't miss this date.
  • Print screen - either by clicking Ctrl-P / Cmd-P or navigating to Menu/Print Screen. Make sure to save as a Web Page. This should save your data as an HTML file.
  • Go to your save's "Upload New Season" page, and upload the HTML document there with some other information about your season. Done!

Caveats

Unfortunately, given that the app needs a specifically-formatted squad-view, there are a few restrictions on who can use the app:

  • Your Football Manager needs to be in English. Internally, the app looks for certain column titles (such as 'Name'), which are only configured for English.
  • For now, this is limited to FM21. There is no equivalent FM18/19/20 squad view, and the app assumes that the xG statistic is part of the squad view. If there is sufficient interest, I can look into configuring this for older versions of FM.
